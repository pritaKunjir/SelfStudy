Q2 Write a java program to print below mentioned pattern.


public class PatternPrinter {
    public static void main(String[] args) {
        int n = 10; // Number of stars to print

        // Print the pattern
        for (int i = 0; i < n; i++) {
            System.out.print("* ");
        }
    }
}


OUTPUT :
                        *

                     *      *

                 *       *      *

              *      *        *      *
